# scDesignPop: Simulator for population-scale single-cell expression RNA-sequencing data

Simulate realistic population-scale single-cell RNA-sequencing
(scRNA-seq) data for single-cell expression quantitative trait loci
analysis (sc-eQTL) using input eQTL data and scRNA-seq data. We use a
flexible, statistical modeling framework that learns interpretable
parameters from real data.
