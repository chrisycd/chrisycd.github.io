# A SingleCellExperiment object containing both cell type and individual id

A subset of the scRNA-seq data from the OneK1K cohort which has been
anonymized.

## Usage

``` r
data("example_sce")
```

## Format

### `example_sce`

A data frame with 100 genes (rows) and 17,918 cells (cols) for 20
individuals :

- cell_type:

  Cell type

- indiv:

  Individual id
